# 🛒 Fresh POS

A lightweight, offline-first POS system built with HTML, CSS, and Vanilla JS.  
Uses **Google Sheets** as a free backend and `localStorage` for fast performance.

---

## ✨ Features

- ⚡ Hybrid cloud sync (local + Google Sheets)
- 🛒 Billing with decimals, discounts & split payments
- 🖨️ 80mm thermal invoice printing
- 📦 Inventory management with stock tracking
- 📊 Sales reports with Chart.js + CSV export
- 📱 Mobile-friendly UI

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Google Apps Script  
- Database: Google Sheets  
- Storage: localStorage
